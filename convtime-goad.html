IRST <-> EST Time Converter

Convert IRST (Iran Standard Time) to EST (Eastern Standard Time)

Convert EST (Eastern Standard Time) to IRST (Iran Standard Time)

How to Use

  • Enter a time as a real number (e.g., 18.33 for 18:33) and click convert.
  • Result is shown as a float and also as hh:mm for clarity.
  • Valid input range: 1.00 to 24.00. Minutes must be two digits after the decimal point.

Examples

  • IRST to EST: 18.33 → 10.03 (10:03 EST)
  • EST to IRST: 1.40 → 10.10 (10:10 IRST)
Note: Conversion does not account for daylight saving time. IRST is always UTC+3:30, EST is always UTC-5:00.
This tool uses the real-number format as specified.
